# 🎮 Tic Tac Toe Game (Java)

## 📌 Description

This is a simple **console-based Tic Tac Toe game** developed using Java.
It allows two players to play the classic game in the terminal.

---

## 🚀 Features

* 👥 Two-player gameplay (X and O)
* ✅ Winner detection
* 🤝 Draw detection
* 🖥️ Simple and clean console interface

---

## 🛠️ Technologies Used

* Java
* VS Code
* Git & GitHub

---

## ▶️ How to Run

1. Compile the program:

```
javac TicTacToe.java
```

2. Run the program:

```
java TicTacToe
```

---

## 🎯 Game Rules

* The game is played on a 3×3 grid
* Players take turns entering row and column (0–2)
* First player to align 3 marks (row, column, diagonal) wins
